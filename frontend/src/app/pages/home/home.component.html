About This Page

This application manages student information through a RESTful API. The main features include:

/students Endpoint

  • Lists all students in a table view
  • Add new students using the Add New Student button
  • Edit existing student information with the Edit button
  • Remove students using the Delete button

Address Information

To view a student's address, click the View Address button next to their record.

Note: The address endpoint is protected with Basic Auth (which is currently only used in the get request)

API Documentation

Endpoint Method Description
/students GET List all students
/students POST Add new student
/students/:id UPDATE Update student
/address/:id GET Get student address (Basic Auth)
